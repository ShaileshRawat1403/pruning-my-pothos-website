## Benefits

- Huge service catalog and global reach.
- Mature IAM and networking primitives.
- Reliable infra for long-running workloads.

## Prerequisites

- Clear account structure.
- A basic VPC mental model.

## Quick snippets

```bash
aws configure
aws sts get-caller-identity
```

## Where it fails

Complexity tax is real; small teams can drown without guardrails.
