## Benefits

- Strong enterprise identity and policy controls.
- Tight integration with Microsoft ecosystems.
- Clear AI service pathways via Azure AI Foundry.

## Prerequisites

- A resource group and region plan.
- Familiarity with Azure identities and roles.

## Quick snippets

```bash
az login
az account show
```

## Resources

- [Azure docs](https://learn.microsoft.com/azure/)
- [Azure CLI](https://learn.microsoft.com/cli/azure/)

## Where it fails

The defaults are locked down; first-time setup can feel slow without a map.
