## Benefits

- Fast feedback without leaving the terminal.
- Composable workflows with scripts and pipes.
- Easy to swap models per task.

## Prerequisites

- Installed CLIs and API keys.
- A basic secrets management habit.

## Quick snippets

```bash
codex --help
```

```bash
claude --help
```

```bash
gemini --help
```
